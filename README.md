# 🛍️ Forever – Full Stack MERN Clothing Store

Forever is a full-stack e-commerce web application built using the MERN stack.  
It provides a complete online shopping experience including authentication, product browsing, cart management, order processing, and Stripe payment integration.

This project demonstrates building RESTful APIs, implementing secure JWT-based authentication, integrating third-party payment services, and preparing a full-stack application for deployment.

---

## 🚀 Features

- User Signup & Login (JWT Authentication)
- Secure Protected Routes
- Product Listing & Category Filtering
- Add to Cart & Cart Management
- Order Placement Workflow
- Stripe Payment Integration (Test Mode)
- Admin Product Management
- Fully Responsive UI

---

## 💳 Payment Integration

Payments are integrated using Stripe (Test Mode) to simulate a secure checkout experience.

---

## 🌍 Deployment

- Frontend deployed using Vercel
- Backend structured for production deployment
- Environment variables configured for security
- Live payments not enabled (business verification requires banking details)
